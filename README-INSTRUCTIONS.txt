INSTALL USING VS CODE
1. Download and unzip project-a-v3-polish.zip.
2. Open your existing Project A repository in VS Code.
3. Open the unzipped folder in your laptop's Files app.
4. Select every file inside the unzipped folder.
5. Drag those files into the PROJECT-A file list on the left side of VS Code.
6. When VS Code asks whether to replace existing files, choose Replace.
7. Confirm that chris-profile.html, style.css and script.js are visible.
8. Open Source Control using the branch icon on the left.
9. Type this commit message: Add polished professional profile
10. Click Commit & Push.
11. Wait 1-2 minutes.
12. Refresh your live site and click Chris Custom Installations > View Profile.
