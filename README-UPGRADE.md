# Project A Upgrade

This is a static front-end MVP demo.

Included:
- Permanent dark theme
- Home page
- Search suggestions
- Professional directory
- Service, city, rating and availability filters
- Favorites saved in browser localStorage
- Sample profiles
- Starting prices
- Trust badges
- Sample reviews
- Request quote modal
- Message modal
- Sample professional signup
- Responsive mobile layout

Important:
- Profiles, ratings and reviews are clearly sample/demo data.
- Forms do not send, store or upload anything.
- Real accounts, databases, messaging, maps, payments, notifications and photo uploads require a backend.

Upload these files to the root of your GitHub repository:
- index.html
- professionals.html
- style.css
- script.js


Version 2 also includes:
- Professional dashboard demo
- Availability editor
- Starting price editor
- Trust-score display
- Quote-request management demo
- Mini profile builder with live preview
- Login page
- Customer/professional signup page

These remain static front-end demos. Authentication, permanent storage, real messages,
real reviews, booking, maps, payment processing, notifications, identity verification,
background checks, and file uploads require a secure backend and third-party services.
