INSTALLATION

1. Download and unzip this folder.
2. Open your current website folder in VS Code.
3. Back up your old files.
4. Delete or replace:
   index.html
   style.css
   script.js
   about.html
   trust-safety.html
   verification-center.html
5. Drag all files from this folder into the same website folder.
6. Right-click index.html and choose Open with Live Server.

IMPORTANT

This is a front-end demo. The quote form, search, verification uploads, Trust Score, and badges are visual demonstrations.

Do not collect real IDs, background-check documents, insurance certificates, or other private information until you add secure user accounts, private cloud storage, encryption, a database, access controls, and appropriate privacy policies.
